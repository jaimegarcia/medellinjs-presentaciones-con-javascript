Agregando bespoke-spotlight - Javascript

main.js

Simplemente se agrega spotlight: true

bespoke.horizontal.from('article', {
    bullets: 'li, .bullet',
    scale: true,
    hash: true,
    progress: true,
    state: true,
    spotlight: true
});
